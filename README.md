ni# Network Security Simulation using Cisco Packet Tracer

## Project Overview
This project demonstrates Network Security using Cisco Packet Tracer. A firewall is implemented using Access Control Lists (ACLs) to allow only authorized users to access the server while blocking unauthorized users.

## Tools Used
- Cisco Packet Tracer
- Router
- Switch
- PCs
- Server

## Features
- Firewall using ACL
- IP Addressing
- Secure Communication
- Network Simulation
